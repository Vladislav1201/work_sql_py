Описание задачи.
С помощью библиотеки psycopg2 была реализована программа включающая информацию о всех городах РБ и дающая возможность получить сведения о них.

Был создан класс DBFacade аналогично тому что на занятии, проинициализирован и соединенный с локальной базой данных ‘belarus’.

Созданы три таблицы: город, достопримечательности и маршруты.

Таблица город включает в себя id, name, region, population, area. 
Эти данные я сам заполнил данными из интернета. Все 115 белорусских городов. Вся информация является достоверной кроме area - она условная.

Таблица достопримечательности включает в себя id, city_id, name, type. 
Табдица марашрут включает в себя di, город из которого строится маршрут, город в который строится маршрут, транспорт на котором можно доехать из одного города в другой, и его расписание.

—Метод добавить все города, проверьте пустая ли таблица и если да, то заполняет ее всеми белорусскими городами. Если все 115 городов есть, то выводится сообщение о том, что уже все добавлено. Сделано с целью того чтобы не добавлялись каждый строчки и таблица не увеличилась многократно.

—Метод просмиотра информации о всех городах

—Метод просмотра информации о всех по конкретной области

—Метод показывающий самые большие города. Пользовтель вводит сколько городов хочет увидеть и ему выводится столько самых больших 

—Метод показывающий топ городов по населению. По аналогии пользователь вввдит желаемое количество и столько городов выводится 

—Метод по которому можно посмотреть информацию о конкретном городе

—Метод добавляющий достопримечательность о городе в таблицу достопримечтельности 

—Метод показывающий все достопричателности города по введенному названию города 

—Метод добавления маршрута между городами в отдельную таблицу

—Метод позволяющий показать по введенному id города а и города б каким транспортном можно добраться из одного в другой 

Сделано удобное меню для вызова на выбор каждого из этих методов и добавлена строчка позволяющая открыть меню повторно только после нажатия любого символа.
